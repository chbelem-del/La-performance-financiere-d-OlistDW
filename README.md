Olist E-commerce Analytics Dashboard:

Description:

Ce projet a été réalisé dans le cadre de mon projet de fin de formation Data Analyst.

L'objectif est de construire une solution décisionnelle permettant d'analyser les performances commerciales d'une plateforme e-commerce (Olist) à travers un tableau de bord interactif développé avec Power BI.

---

Objectifs:

* Analyser les ventes de la plateforme
* Suivre les performances des produits
* Évaluer les délais de livraison
* Analyser les paiements et la satisfaction client
* Construire un tableau de bord interactif pour faciliter la prise de décision

---

Technologies utilisées:

* Python (Pandas)
* VS Code
* Power BI
* DAX
* Git & GitHub

---

Dataset:

Le projet utilise le dataset public **Olist Brazilian E-commerce Dataset**, comprenant des informations sur :

* Commandes
* Clients
* Produits
* Vendeurs
* Paiements
* Avis clients
* Géolocalisation

---

Pipeline du projet:

1. Importation des données
2. Analyse exploratoire (EDA)
3. Nettoyage des données
4. Fusion des tables
5. Création des dimensions et des tables de faits
6. Modélisation en étoile
7. Création des mesures DAX
8. Développement du dashboard Power BI

---

Principaux KPI:

* Chiffre d'affaires
* Nombre de commandes
* Nombre de clients
* Nombre de vendeurs
* Note moyenne
* Délai moyen de livraison
* Répartition des paiements

---

Dashboard:

Le dashboard est composé de deux pages :

Page 1 – Vue d'ensemble

* KPI principaux
* Évolution du chiffre d'affaires
* Chiffre d'affaires par catégorie
* Chiffre d'affaires par État
* Répartition des paiements

Page 2 – Analyse détaillée

* Analyse des clients et vendeurs
* Carte géographique
* Délais moyens de livraison
* Évolution des commandes

---

Résultats:

Ce projet permet d'identifier :

* les catégories les plus performantes ;
* les États générant le plus de chiffre d'affaires ;
* les moyens de paiement les plus utilisés ;
* les performances logistiques ;
* les principaux indicateurs de performance (KPI).
